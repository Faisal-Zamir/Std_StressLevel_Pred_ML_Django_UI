# Student Stress Level Prediction | Machine Learning Project with Django UI

## Description
A machine learning project that predicts student stress levels and provides a user-friendly interface built with Django.

## Features
- Predict student stress level score based on input features.
- Web-based UI using Django.
- Interactive and user-friendly input forms.
- Displays predictions along with model accuracy.

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Faisal-Zamir/Std_StressLevel_Pred_ML_Django_UI.git
```
2. Navigate to the project directory:

```bash
cd Std_StressLevel_Pred_ML_Django_UI
```
3. Install required packages:
```bash
pip install -r requirements.txt
```
4. Run the Django server:
```bash
python manage.py runserver
```
5. Open your browser at:
```bash
http://127.0.0.1:8000
```
## Usage

To use the Student Performance Prediction application:

- **Step 1:** Fill in student details in the input form.
- **Step 2:** Click the **"Predict Stress Level"** button.
- **Step 3:** View the predicted Stress Level along with related metrics such as the Confidence Level in %.

The model is trained on a publicly available dataset from Kaggle:
